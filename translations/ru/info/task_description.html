В качестве входных данных вы получите формулу химического вещества и число N. Вам необходимо будет составить список всех химических элементов, количество атомов которых в данном веществе >= N.
Обратите внимание, что для записи некоторых веществ будут использоваться скобки - () и [], которые нужно учитывать при подсчете. Эта статья может вам помочь.

Входные данные: Химическая формула и количество атомов.

Выходные данные: Список химических элементов.

{{init_code_tmpl}}

Как это используется: Для химического анализа.

Предусловия:
1 <= количество различных элементов <= 10
